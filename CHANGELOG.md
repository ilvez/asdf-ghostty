# Changelog
## 1.2.0 (2026-03-09)
- Build from officially bundled sources
- Check source bundle signatures with minisign
- Populate GHOSTTY_RESOURCES_DIR correctly for themes etc..

- Use `-fsys=fontconfig` flag for better font handling
- Test zig version before build

- Basic Linux support
- CI/CD pipeline
- Linting
